# Pixel Buffer Test
Simple performance test comparing pixel write speed (32-bit words) to buffers in host memory (RAM) vs VRAM (eDRAM).Each buffer is set as framebuffer in turn, and performance is measured via FPS. Press Triangle to switch between memory targets.
